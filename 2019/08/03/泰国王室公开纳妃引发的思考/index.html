泰国王室公开纳妃引发的思考

发表于 2019-08-03 | 分类于 随笔

永乐大帝出了一期关于时空旅行的科普视频，里面提到，一个体积无限小，密度无限大的物质会使周围的时空发生弯曲，蕴含极大的能量，状态极其”不稳定”，比如黑洞

泰国王室公开纳妃，87年来首次承认一夫多妻制；

我突然意识到（虽然我以前也意识到，但从来“没有”这次意识那么深刻，引发这么多思考，足够重视），

这感觉就像 荀子说的：”吾尝终日而思矣，不如须臾之所学也“一样；

对于现在的时空而言：

如果地球是个小宇宙，各国领导人就是那个体积无限小，质量无限大的天体，

如果中国是个小宇宙，北上广杭深就是那个体积无限小，质量无限大的天体，

如果行业是个小宇宙，数字信息业就是那个体积无限小，质量无限大的天体，

他们的共性是状态及其不稳定，无论是正稳定还是负稳定（正为积极、负为消极）

归纳一共有四个重要的概念：

  • 能量大
  • 能量小
  • 正稳定
  • 负稳定

进行组合后为：

  • 能量大 + 正稳定
  • 能量大 + 负稳定
  • 能量小 + 正稳定
  • 能量小 + 负稳定

具体表现：

type：国家

  • 能量大 + 正稳定 => 五常、欧盟、德日韩 ( 国际地位，科技，货币 )
  • 能量大 + 负稳定 => 二战时的法西斯国家 ( 战争 )
  • 能量小 + 正稳定 => 荷兰、瑞士、丹麦 ( 科技，单一支柱产业)
  • 能量小 + 负稳定 => 伊朗、伊拉克、巴基斯坦 ( 战争 )

type：中国

  • 能量大 + 正稳定 => 北上广杭深 ( 工业，制造业，信息业 )
  • 能量大 + 负稳定 => 无
  • 能量小 + 正稳定 => 鲁，豫，黑 ( 农业 )
  • 能量小 + 负稳定 => 近期的香港

type：行业

  • 能量大 + 正稳定 => 工业，制造业，信息业
  • 能量大 + 负稳定 => 黑产
  • 能量小 + 正稳定 => 农业
  • 能量小 + 负稳定 => 灰产

如果想要在能量大且正稳定的天体得到好处，有个重要前提条件：打造一个坚固的飞船，否则，不仅捞不到好处，还会被强大的引力撕的粉身碎骨；

相信环境的力量，但更重要的是要有B数
